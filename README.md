# Тетрис

Классический тетрис в одном HTML-файле — открывается в любом браузере, ничего устанавливать не нужно.

## Как играть

Открой файл [`tetris.html`](tetris.html) двойным щелчком (или перетащи в окно браузера) и нажми **▶ Старт**.

Управление:

| Клавиша | Действие |
|---|---|
| ← → | двигать фигуру |
| ↑ | повернуть |
| ↓ | ускорить падение |
| Пробел | мгновенный сброс |
| P | пауза |

## Возможности

- Счёт, уровни и растущая с уровнем скорость
- «Мешок из 7» + подбор фигур с учётом состояния поля
- Анимация схлопывания линий, частицы, тряска экрана, звук
- Таблица рекордов, сохраняется между запусками

## Служебные файлы

- `server.ps1`, `.claude/launch.json` — локальный сервер для отладки, для игры не нужны.
